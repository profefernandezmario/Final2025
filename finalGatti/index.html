Sobre el proyecto

Esto es un pequeño proyecto de un sitio web compuesto por diferentes páginas que enstan enlazadas entre sí. Es una práctica para la construcción de sitios web con estructuras comunes y una correcta navegación con estilos simples.

El objetivo principal es el dominio del uso de un HTML semántico, CSS y Bootstrap para armar una web responsiva y que pueda mantener una apariencia profesional, adaptativa y coherencia con los distintos dispositivos.
